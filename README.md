# BNS 2018

Team BNS's 2018 VEX U robot code. Written in C and run on PROS.

## Getting Started

From the command line:
```
$ git clone https://github.com/jtkiesel/jankyBoi
```

### Prerequisites

[PROS](https://pros.cs.purdue.edu)
